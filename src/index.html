❓ Aide — SearchFlow

🔍 À quoi sert SearchFlow ?

SearchFlow vous permet de rechercher du texte dans tous vos fichiers d'un dossier, rapidement et facilement. Contrairement à la recherche Windows qui cherche uniquement les noms de fichiers, SearchFlow lit l'intérieur de chaque fichier pour trouver exactement ce que vous cherchez.

📁 Dossier à analyser

Cliquez sur Parcourir pour sélectionner le dossier, ou glissez-déposez directement un dossier depuis l'explorateur Windows dans la zone pointillée. SearchFlow analysera ce dossier et tous ses sous-dossiers automatiquement.

✏️ Champ de recherche

Tapez le ou les mots que vous recherchez. Vous pouvez appuyer sur Entrée pour lancer la recherche sans cliquer sur le bouton.

⚙️ Modes de recherche

Le menu à côté du champ de recherche permet de choisir comment SearchFlow interprète ce que vous tapez.

Contient tous les mots

Le mode le plus utile au quotidien. Il trouve les lignes qui contiennent tous les mots tapés, peu importe l'ordre, et même s'il y a d'autres mots entre eux. Exemple : taper jean dupont trouve aussi bien « Dupont Jean » que « Jean est l'ami de Dupont ».

Contient au moins un mot

Trouve les lignes qui contiennent au moins un des mots tapés. Pratique quand vous hésitez sur le terme exact : facture devis trouve les lignes avec « facture » ou « devis ».

Contient la phrase exacte

Trouve les lignes contenant exactement ce que vous avez tapé, dans le même ordre. Exemple : jean dupont ne trouvera que « jean dupont », pas « Dupont Jean ».

Recherche avancée (regex)

Pour les utilisateurs expérimentés : des motifs avec expressions régulières. Exemple : facture.*2024 trouve « facture » suivi de n'importe quoi puis « 2024 ». Si vous ne savez pas ce qu'est une regex, vous n'en avez pas besoin : utilisez un des trois modes ci-dessus.

🔧 Options

Respecter la casse — si activé, « Jean » et « jean » sont considérés comme différents. Par défaut (case décochée), la recherche ignore les majuscules/minuscules.

Ignorer les accents — si activé, les lettres accentuées sont traitées comme leur version simple : « e », « é », « è », « ê » deviennent équivalents, tout comme « c » et « ç ». Concrètement, taper jerome trouvera aussi « Jérôme », et francois trouvera « François ». Très pratique pour chercher un nom sans se soucier des accents. Laissez la case décochée pour une recherche stricte (c'est aussi un peu plus rapide).

Inclure les sous-dossiers — si activé, SearchFlow parcourt aussi tous les sous-dossiers du dossier sélectionné.

📄 Types de fichiers

Utilisez le toggle pour tout cocher ou tout décocher. Vous pouvez aussi cocher ou décocher manuellement chaque extension. Décocher des types inutiles accélère la recherche.

📊 Barre de progression

Pendant la recherche, une barre de progression s'affiche avec le nombre de fichiers analysés en temps réel et le nombre de résultats trouvés jusqu'à présent.

📋 Résultats

Chaque résultat affiche le nom du fichier, son chemin complet, le numéro de ligne et le contenu de la ligne avec les mots recherchés surlignés en orange.

Clic gauche sur un résultat → copie le chemin du fichier.

Clic droit sur un résultat → menu avec options :

  • Ouvrir le fichier — ouvre le fichier avec son application par défaut
  • Ouvrir le dossier contenant — ouvre l'explorateur Windows à l'emplacement du fichier
  • Copier le chemin — copie le chemin complet dans le presse-papier

💾 Export des résultats

Après une recherche, cochez les résultats que vous souhaitez exporter puis cliquez sur Exporter TXT ou Exporter CSV. Une fenêtre s'ouvre pour choisir où sauvegarder le fichier. Le fichier CSV peut être ouvert directement dans Excel.

⏱️ Historique

SearchFlow mémorise vos 10 dernières recherches. Cliquez sur une recherche récente pour la relancer automatiquement. Vous pouvez désactiver l'historique ou supprimer des entrées individuellement.

⚠️ Avertissement fichiers volumineux

Si vous tentez d'ouvrir un fichier de plus de 50 Mo, SearchFlow vous avertit avant de l'ouvrir pour éviter tout ralentissement de votre PC.
